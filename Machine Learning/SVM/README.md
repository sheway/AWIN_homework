# 支援向量機(Support Vector Machine, SVM)

## 訓練模型
藉由訓練集資料取出 80% 當作訓練資料，使用 SVM 的兩種 state(linear, poly)，準確率如下：

- SVM 使用 linear state 的 Accuracy 為 0.9625。
- SVM 使用 poly state 的 Accuracy 為 0.9725。
